树莓派配置 USB 无线网卡

5月 08, 2016 发布在 Linux笔记

树莓派配置 USB 无线网卡来上网的过程。

本人使用的USB无线网卡型号：EP-N8508GS（树莓派专用型号）

一、 检查 USB 无线网卡是否已经正确识别

将无线 USB 网卡插入树莓派后启动树莓派，比较不建议热插拔，因为插入的一瞬间会有比较高的电流，如果电源输出不够可能导致树莓派重启。
进入 shell 界面后输入命令 lsusb
