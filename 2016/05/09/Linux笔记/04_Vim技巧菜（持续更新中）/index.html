5月 09, 2016 发布在 Linux笔记

Vim 常用命令整理，提高生产效率。

VIM快速移动

在移动光标前请先按下 ESC 键，然后 shift+$ 是移动到行尾，0 是移动到行首

VIM查找

命令模式下，输入 /word 后回车，即查找 word，
按 n 查找下一个匹配单词，按 N 查找上一个匹配单词。

修改主题颜色

打开一个Vim窗口，输入命令 :color 或 :colorscheme 后回车查看当前的颜色主题， 可以看到当前的颜色主题为 default。
Vim 的颜色主题文件放在 Vim 运行目录下的 color 目录下，所以我们首先需要知道 Vim 的运行目录。
在 Vim 中输入命令 :echo $VIMRUNTIME 来查看Vim的运行目录

进入 Vim 的运行目录，查看 colors 目录下以 .vim 为结尾的文件， 这些文件即是颜色主题文件，文件名就是主题名字。

打开 /etc/vim/vimrc 文件，在其中加入一行 colorscheme 颜色主题名字，之后保存更改即可。

如 colorscheme blue
